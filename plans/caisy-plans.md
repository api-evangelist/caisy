# Caisy Plans

Caisy offers three tiers designed to scale from individual developers to large enterprises. All plans include access to the GraphQL content API; limits vary by tier.

## Free Plan — $0/month (Always free)

Entry-level plan with no credit card required.

- Users: 3
- Projects: 3
- Locales: 2
- Content Entries: 5,000
- Blueprints (content models): 30
- API Calls: 1 million/month
- Bandwidth/Traffic: 100 GB/month
- API Rate Limit: 5 requests/second
- Image optimization included
- Webhooks included
- Community support

## Growth Plan — $49/month

Designed for growing teams and production projects.

- Users: 12
- Locales: 4
- Content Entries: 20,000
- Blueprints: 60
- API Calls: 10 million/month
- Bandwidth/Traffic: 500 GB/month
- API Rate Limit: 35 requests/second
- Everything in Free, plus:
  - Extended content entry limits
  - Higher API throughput
  - Email support

### Growth Overage Pricing

- Additional user: $10/month each
- Additional 10,000 entries: $20/month
- Additional locale: $15/month
- Additional 1 TB bandwidth: $60/month
- Additional 10 blueprints: $10/month

## Enterprise Plan — Starting at $1,499/month

Custom capacity for large organizations and high-traffic deployments.

- Users: Custom quota
- Locales: Custom quota
- Content Entries: Custom quota
- Blueprints: Custom quota
- API Calls: Custom quota
- Bandwidth/Traffic: Custom quota
- API Rate Limit: 50+ requests/second (customizable)
- Everything in Growth, plus:
  - Custom user roles
  - Organization-level audit logs
  - Content workflows
  - Custom SSO (Single Sign-On)
  - SLA with 99.9% uptime guarantee
  - Dedicated account manager
  - On-premise hosting option
  - Custom domain

Payments are processed via Stripe (credit card).

Source: https://caisy.io/pricing
